Download .zip

  1. Download and extract the .zip file
  2. Run FerrisPad.exe
  3. Optionally, create a desktop shortcut
‚ö†Ô∏è Windows Security Warning

When running FerrisPad for the first time, Windows may show a "Windows protected your PC" warning. This is normal for open source software not signed with a commercial certificate (costs $100-400/year).

FerrisPad is safe to run. The source code is open and auditable on GitHub.

To run FerrisPad:

  1. Click "More info" on the warning dialog
  2. Click "Run anyway"

Alternative: Right-click the .zip file ‚Üí Properties ‚Üí Check "Unblock" ‚Üí OK ‚Üí Extract and run.

This warning appears for many open source projects like GIMP, Audacity, and other community software.

macOS

macOS 11+ (Intel & Apple Silicon)

Download .dmg
Installation Instructions
  1. Download and open the .dmg file
  2. Drag FerrisPad.app to Applications folder
  3. Launch from Applications or Spotlight
  4. If blocked, go to System Settings > Privacy & Security and click "Open Anyway"

Linux

Ubuntu/Debian-based distros

Download .deb
Installation Instructions
  1. Download the .deb file
  2. Install with: sudo dpkg -i FerrisPad-v0.1.4-ubuntu-amd64.deb
  3. Or double-click the file to install via Software Center
  4. Launch from Applications menu or run FerrisPad

Compile from Source

Build FerrisPad yourself in just a few steps

1

Install Prerequisites

# Install Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Install FLTK dependencies
sudo apt-get install libfltk1.3-dev libfontconfig1-dev libxext-dev \
  libxft-dev libxinerama-dev libxcursor-dev libxrender-dev \
  libxfixes-dev libpango1.0-dev libgl1-mesa-dev libglu1-mesa-dev
# Install Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Install Xcode Command Line Tools
xcode-select --install
# Install Rust from https://rustup.rs/
# Download and run rustup-init.exe

# FLTK should work out of the box on Windows
2

Clone and Build

# Clone the repository
git clone https://github.com/fedro86/ferrispad
cd ferrispad

# Build in release mode
cargo build --release

# The binary will be at: ./target/release/FerrisPad
3

Run FerrisPad

# Run directly
cargo run --release

# Or use the compiled binary
./target/release/FerrisPad

# Optional: Install desktop integration (Linux only)
./install-desktop.sh
