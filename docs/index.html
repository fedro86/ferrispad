‚ö†Ô∏è Windows Security Warning

When running FerrisPad for the first time, Windows may show a "Windows protected your PC" warning or Windows Defender may quarantine the executable. This is normal for open source software not signed with a commercial certificate (costs $100-400/year).

FerrisPad is safe to run. The source code is open and auditable on GitHub.

If you see "Windows protected your PC" warning:

  1. Click "More info" on the warning dialog
  2. Click "Run anyway"

If Windows Defender deletes the executable:

  1. Open Windows Security (search in Start menu)
  2. Go to Virus & threat protection ‚Üí Manage settings
  3. Scroll down to Exclusions ‚Üí Click Add or remove exclusions
  4. Click Add an exclusion ‚Üí Choose Folder
  5. Select the folder where you extracted FerrisPad
  6. Re-extract the .zip file to restore the executable

Alternative (before extraction):

Right-click the .zip file ‚Üí Properties ‚Üí Check "Unblock" ‚Üí OK ‚Üí Extract and run.

This warning appears for many open source projects like GIMP, Audacity, and other community software.

macOS

macOS 11+ (Intel & Apple Silicon)

Download .dmg
Installation Instructions
  1. Download and open the .dmg file
  2. Drag FerrisPad.app to Applications folder
  3. Launch from Applications or Spotlight
‚ö†Ô∏è macOS Security Warning

When opening FerrisPad for the first time, macOS may show a warning that the app "cannot be opened because it is from an unidentified developer" or "cannot be opened because Apple cannot check it for malicious software." This is normal for open source software not signed with an Apple Developer certificate (costs $99/year).

FerrisPad is safe to run. The source code is open and auditable on GitHub.

To open FerrisPad:

Method 1 (Terminal - Most Reliable):

  1. Open Terminal
  2. Run: xattr -cr /Applications/FerrisPad.app
  3. This removes the quarantine flag
  4. Then open FerrisPad normally from Applications

Method 2 (System Settings):

  1. Try to open FerrisPad normally (it will be blocked)
  2. Go to System Settings ‚Üí Privacy & Security
  3. Scroll down to the Security section
  4. Click "Open Anyway" next to the FerrisPad message
  5. Click "Open" to confirm

Method 3 (Right-click - May Not Work):

  1. Right-click (or Control-click) FerrisPad.app in Applications
  2. Select "Open" from the menu
  3. If an "Open" button appears in the dialog, click it
  4. Note: This doesn't work reliably on newer macOS versions

This warning appears for many open source projects like GIMP, Audacity, Blender, and other community software that aren't code-signed by Apple.

Privacy & Updates

Stay up-to-date while maintaining full control

üîî

Auto-Update Check

FerrisPad can automatically check for new versions once per 24 hours by querying GitHub's public API. You'll see a subtle notification banner when updates are available.

üîí

Privacy-First Design

  • No telemetry - We don't collect usage data, analytics, or personal information
  • No tracking - No cookies, no user identification, no behavioral monitoring
  • GitHub API only - Only checks public release information
  • Fully optional - Can be completely disabled in Settings
  • Transparent - All code is open source and auditable
‚úã

You're in Control

Go to File ‚Üí Settings to disable auto-update checks entirely. You can always manually check for updates via Help ‚Üí Check for Updates. FerrisPad will never download or install anything without your explicit permission.

Following best practices from VS Code, Firefox, and Notepad++, FerrisPad respects your privacy while keeping you informed.

Compile from Source

Build FerrisPad yourself in just a few steps

1

Install Prerequisites

# Install Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Install FLTK dependencies
sudo apt-get install libfltk1.3-dev libfontconfig1-dev libxext-dev \
  libxft-dev libxinerama-dev libxcursor-dev libxrender-dev \
  libxfixes-dev libpango1.0-dev libgl1-mesa-dev libglu1-mesa-dev
# Install Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Install Xcode Command Line Tools
xcode-select --install
# Install Rust from https://rustup.rs/
# Download and run rustup-init.exe

# FLTK should work out of the box on Windows
2

Clone and Build

# Clone the repository
git clone https://github.com/fedro86/ferrispad
cd ferrispad

# Build in release mode
cargo build --release

# The binary will be at: ./target/release/FerrisPad
3

Run FerrisPad

# Run directly
cargo run --release

# Or use the compiled binary
./target/release/FerrisPad

# Optional: Install desktop integration (Linux only)
./install-desktop.sh
